# Example Flows

These tracked examples are intended to show off the node catalog without mixing demo content into the user's local `flows/` directory.

Load them from the editor with the `Examples` button in the toolbar.

Included examples:

- `support-triage-router` - prompt + LLM + conditional routing
- `local-note-reviewer` - file read, review, and writeback
- `workspace-inventory-report` - shell, code, merge, and file output
- `iterative-headline-polisher` - loop-driven refinement
- `meeting-copilot-chat` - UI interface plus persistent conversation memory
- `knowledge-base-bootstrap` - embed notes into a local vector collection
- `grounded-context-retriever` - semantic retrieval from a local vector collection
- `research-brief-with-review` - map-reduce fetch, agent synthesis, human approval, and file save
