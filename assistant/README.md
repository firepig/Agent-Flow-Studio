This directory is local runtime state for the built-in agent tooling.

Typical files created here:

- `persona.json`
- `memory_layers.json`
- `chat_history.json`
- `conversation.json`
- `conversation_log.md`
- `thought_journal.jsonl`
- `consolidation_log.jsonl`

These files are intentionally ignored so personal or generated data does not get committed.
